# Radio

Control de selección exclusiva dentro de un grupo. Al compartir el mismo `name`, los radios son mutuamente excluyentes: seleccionar uno deselecciona el resto. Utiliza el elemento `<input type="radio">` nativo — no necesita Radix, ya es accesible por defecto.

## Uso

```tsx
import { Radio } from '@studiolxd/brand/radio';

<Radio name="plan" value="basico"   aria-label="Básico"    defaultChecked />
<Radio name="plan" value="pro"      aria-label="Pro" />
<Radio name="plan" value="empresa"  aria-label="Empresa" />
```

Todos los radios del mismo grupo deben compartir el atributo `name`. El `value` identifica la opción seleccionada al enviar el formulario. Para uso controlado, pasa `checked` y `onChange`.
