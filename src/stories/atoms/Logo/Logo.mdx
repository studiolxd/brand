# Logotipo

El logotipo SVG hereda el color del contexto mediante `currentColor`. Sobre fondo oscuro adapta el color automáticamente cuando está dentro de un contenedor `.surface-dark`. La prop `dark` aplica directamente la variante clara cuando no hay contexto de superficie disponible.
