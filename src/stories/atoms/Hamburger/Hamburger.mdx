# Hamburger

Botón de tres barras que alterna la visibilidad de la navegación móvil. Anima a forma de X mediante CSS cuando `aria-expanded="true"`.

## Theming oscuro

El Hamburger no necesita prop `dark` — adapta el color de las barras automáticamente cuando un ancestro tiene la clase `.surface-dark`. No hay portal Radix involucrado, por lo que la cascade CSS funciona de forma normal.
