# Chevron

Icono SVG decorativo con forma de ángulo (›). Se usa como indicador de expansión, navegación o dirección en selectores, acordeones y menús. Siempre se renderiza con `aria-hidden="true"` para no ser anunciado por lectores de pantalla.

## Tamaños

Tres tamaños disponibles: `sm`, `md` (por defecto) y `lg`. El ancho se controla mediante tokens CSS y es aproximadamente la mitad de ancho que `Arrow` para el mismo nivel de `size`.
