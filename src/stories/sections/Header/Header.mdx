# Header

Organismo de navegacion principal. Compone el atomo Logo, el atomo Hamburger y una barra de navegacion responsive.

En mobile (< 1440px), la navegacion se presenta como overlay a pantalla completa que se despliega desde arriba al pulsar el hamburger. En desktop (>= 1440px), el hamburger se oculta y la navegacion se dispone en horizontal.
