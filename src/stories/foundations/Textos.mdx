# Textos

Estilos base para los elementos de texto en contexto de lectura: enlaces, párrafos, encabezados y listas.

---

## Enlace

Los enlaces heredan el color del texto circundante. En reposo muestran subrayado; en hover lo ocultan.

### Tokens

| Token | Valor |
|---|---|
| `--link-color` | `inherit` |
| `--link-text-decoration` | `underline` |
| `--link-text-decoration-hover` | `none` |
| `--link-cursor` | `pointer` |
