# ProjectCard

Tarjeta de proyecto completamente clicable. Muestra categoría (tag con variante de color), título, imagen, descripción breve y un CTA visual. Es la unidad base reutilizada tanto en `ProjectCarousel` como en `ProjectGrid`.

El texto del CTA se controla con la prop `ctaLabel` (por defecto "Leer más"), lo que permite inyectar traducciones desde la aplicación consumidora.
