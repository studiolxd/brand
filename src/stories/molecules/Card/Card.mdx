# Card

Tarjeta de servicio completamente clicable con color de fondo, título, descripción y flecha indicadora. El CTA (`ctaLabel`) es accesible para lectores de pantalla mediante `VisuallyHidden`, sin duplicar texto visible.

Disponible en cuatro variantes de color (accent-1, accent-2, support-1, support-2) que mapean a la paleta de identidad.
